Configure Cloudflare DNS

Oleh Zarif Fathurrahman Rani
Diterbitkan 2022-06-09 1 min baca

Buat acount dan login ke Cloudflare DNS

Layari Cloudflare DNS dan buat account anda.

Log masuk dan click pada Domain dan Register New Domain

Masukkan domain anda

Tukar Name Server daripada DNS provider lama anda ke Cloudflare.

  1. Login ke Domain Name Provider anda.
  2. Pergi ke nameserver setting.
  3. Masukkan nameserver baharu.

Tunggu sehingga anda dapat confermation email daripada Cloudflare.

Tahniah, anda telah berjaya. Boleh start configure DNS baharu anda.
